# SebasPortfolio - Portfolio Personal

Portfolio web desarrollado con React y Vite que muestra una presentación personal, habilidades principales y secundarias, experiencia laboral, formación académica y un apartado de contacto. El proyecto está organizado en secciones claras y estructuradas para ofrecer una visión general de mi perfil como desarrollador web.

## Secciones del Portfolio

- Presentación inicial  
- Imagen y breve descripción personal  
- Habilidades principales  
- Habilidades secundarias  
- Experiencia laboral  
- Formación académica  
- Información de contacto  

## Tecnologías Utilizadas

- React  
- Vite  
- JavaScript  
- CSS  

## Estructura del Proyecto

El portfolio se compone de componentes independientes que representan cada sección, integrados en una única página para ofrecer una navegación sencilla y una experiencia fluida.

## Ejecución en Local

1. Instalar dependencias:
   
     npm install

   
3. Ejecutar en modo desarrollo:
   
     npm run dev
